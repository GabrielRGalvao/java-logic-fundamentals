# Java Programming Mastery: Logic & Algorithms ☕🚀

Este repositório centraliza meus estudos sobre **Lógica de Programação** e **Algoritmos**. O objetivo é consolidar a base necessária para o desenvolvimento de sistemas backend robustos.

## 🎯 Foco do Aprendizado
Atualmente, estou explorando os fundamentos da linguagem, utilizando desafios técnicos totalmente voltados para prática e solução de problemas aplicando os conceitos de:

- **Estruturas de Dados:** Manipulação de arrays unidimensionais (Vetores) e bidimensionais (Matrizes).
- **Processamento:** Lógica matemática, manipulação de variáveis e precisão decimal.
- **Fluxo de Controle:** Estruturas condicionais e laços de repetição complexos.
- **Clean Code:** Organização de código e nomenclatura de variáveis.

## 🛠️ Stack Tecnológica
- **Linguagem:** Java 21.
- **Ferramentas:** Eclipse IDE, Git/GitHub.
- **Ambiente:** Linux (Ubuntu).

## 📂 Organização do Repositório
O projeto está dividido por módulos de conhecimento:

* `/Estrutura_Sequencial`: Fundamentos, entrada/saída e operadores.
* `/Estrutura_Condicional`: Tomada de decisão e lógica booleana.
* `/Estruturas_Repetitivas`: Otimização de processos com `for`, `while` e `do-while`.
* `/Vetores`: Armazenamento e manipulação de conjuntos de dados em memória.
* `/Matrizes`: Processamento de tabelas e dados multidimensionais.
